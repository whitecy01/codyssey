# 🧠 개발자 성장을 위한 전체 학습 레포지토리
이 레포지토리는 개발자로 성장하기 위한 전반적인 학습 기록과 스터디 정리 공간입니다.
혼자 공부한 내용부터 여러 명이 함께 진행한 스터디, 그리고 기술 문서(RFC) 분석까지 지식이 축적되는 학습 아카이브를 목표로 구성했습니다.

이 저장소는 크게 아래와 같은 학습 영역으로 구성되어 있습니다.

- 도서 기반 개인 학습 정리
